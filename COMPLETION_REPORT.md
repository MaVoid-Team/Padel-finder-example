### How It Works
1. User selects court (4 available)
2. User picks date (calendar)
3. System shows available slots (90-min)
4. User fills booking form
5. System validates form (3-tier)
6. Booking saved to localStorage
7. Success confirmation shown
8. Data persists across sessions
